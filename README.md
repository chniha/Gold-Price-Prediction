# Gold Price Prediction using Machine Learning

## Project Overview
This project predicts Gold Prices using Machine Learning techniques in Python.

## Dataset Features
- Date
- SPX
- USO
- SLV
- EUR/USD
- GLD (Target Variable)

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Model
- Random Forest Regressor

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Correlation Analysis
4. Data Visualization
5. Train-Test Split
6. Model Training
7. Prediction
8. Performance Evaluation

## Results
The Random Forest Regressor model was trained to predict gold prices based on market indicators and achieved good prediction accuracy.
